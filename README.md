# MKD Manager

MKD Manager is a software application designed to manage MKD (Medical Knowledge Database) systems.

## Project Structure

The project consists of the following main components:

- **mkd_manager/**: Main application directory
  - **config/**: Configuration files
  - **core/**: Core application logic
  - **data/**: Data handling modules
  - **database/**: Database management
  - **gui/**: Graphical user interface components
  - **models/**: Data models
  - **rules/**: Business rules
  - **vsn5386r/**: Version-specific components
  - **main.py**: Main application entry point
  - **requirements.txt**: Python dependencies

## Setup

To set up the project, ensure you have Python installed and run:

```bash
pip install -r mkd_manager/requirements.txt
```

## Usage

Run the application using:

```bash
python mkd_manager/main.py
```
