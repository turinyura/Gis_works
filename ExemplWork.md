Обеспечение:

- ✅ **Оснащения рабочего места диспетчера** необходимыми техническими средствами

- ✅ **Взаимодействия с экстренными службами** (МЧС, РСО, Горгаз, Скорая)

### 3. СОСТАВ РАБОТ

#### **3.1. Организационное обеспечение АДС:**

| Направление | Содержание работ | Периодичность |
|------------|-----------------|--------------|
| **Штатное расписание** | Формирование штата диспетчеров, утверждение графиков смен, назначение ответственных | Ежегодно/по необходимости |
| **Регламенты и инструкции** | Разработка и актуализация регламентов работы, скриптов общения, алгоритмов действий при авариях | По мере изменения нормативов |
| **Договорная база** | Заключение договоров с подрядчиками на ТО оборудования, связь, IT-поддержку | По срокам договоров |
| **Лицензирование/допуски** | Оформление допусков персонала к работе, проверка квалификационных удостоверений | 1 раз в год |

#### **3.2. Техническое оснащение и ТО оборудования:**

| Оборудование | Состав работ по содержанию | Периодичность |
|-------------|---------------------------|--------------|
| **Диспетчерский пульт/CRM-система** | Обновление ПО, резервное копирование, техническая поддержка, устранение сбоев | Постоянно/по регламенту |
| **Средства связи (IP-телефония, рации)** | Проверка качества связи, замена оборудования, оплата услуг связи | Ежемесячно/ежеквартально |
| **Система записи разговоров** | Контроль объема памяти, архивирование, проверка работоспособности | Еженедельно |
| **Система SMS/Email-уведомлений** | Настройка шаблонов, мониторинг доставки, оплата услуг | Постоянно |
| **Компьютеры и периферия** | ТО, обновление, замена неисправного оборудования | По графику ТО |
| **Источник бесперебойного питания (ИБП)** | Проверка батарей, тестирование переключения, замена АКБ | 1 раз в квартал |
| **Средства пожаротушения в помещении АДС** | Проверка срока годности, перезарядка, замена | 1 раз в год |

#### **3.3. Работа с персоналом:**

| Направление | Содержание работ | Периодичность |
|------------|-----------------|--------------|
| **Подбор и обучение** | Найм диспетчеров, первичное обучение, стажировка | По необходимости |
| **Инструктажи** | Вводный, первичный, повторный, внеплановый инструктажи по охране труда и регламентам | 1 раз в квартал |
| **Проверка знаний** | Аттестация диспетчеров на знание регламентов, нормативов, скриптов | 1 раз в год |
| **Медицинские осмотры** | Предварительные и периодические медосмотры | 1 раз в год |
| **Мотивация и контроль** | Оценка KPI, премирование, дисциплинарные меры | Ежемесячно |

#### **3.4. Документационное обеспечение:**

| Документ | Содержание работ | Периодичность |
|----------|-----------------|--------------|
| **Журналы учета** (заявок, аварий, инструктажей) | Ведение, пронумеровка, прошивка, хранение | Постоянно |
| **Регламенты и инструкции** | Разработка, согласование, утверждение, актуализация | По мере изменений |
| **Отчетность** | Формирование ежемесячных/квартальных отчетов по работе АДС | Ежемесячно/ежеквартально |
| **Архивирование** | Сохранение записей разговоров, актов, журналов в соответствии со сроками хранения | Постоянно |

|--------|-----------------------------------|--------------|

### 7. ЭТАПЫ ПРОВЕДЕНИЯ (ПОШАГОВАЯ ИНСТРУКЦИЯ)

#### **ЭТАП 1: Планирование и организация (ежегодно/по необходимости)**

```
□ Утвердить штатное расписание АДС с учетом нагрузки и нормативов
□ Разработать/актуализировать регламенты работы, скрипты, инструкции
□ Заключить/продлить договоры с подрядчиками (связь, IT, ТО оборудования)
□ Сформировать график смен диспетчеров с учетом ТК РФ
□ Назначить ответственного за работу АДС приказом по УК
□ Утвердить бюджет на содержание АДС на год
```

#### **ЭТАП 2: Ежедневное обеспечение работы (каждая смена)**

**Шаг 1. Начало смены:**
```
□ Провести пересменку: передача дежурства, информирование о текущих авариях
□ Проверить исправность оборудования (компьютер, телефон, CRM, запись)
□ Протестировать связь с выездными бригадами и экстренными службами
□ Проверить наличие канцелярии, журналов, инструкций на рабочем месте
□ Зафиксировать начало дежурства в журнале
```

**Шаг 2. Работа в течение смены:**
```
□ Принимать звонки/обращения в соответствии со скриптами
□ Регистрировать заявки в CRM с фиксацией времени
□ Классифицировать и назначать исполнителей
□ Контролировать исполнение заявок, эскалировать при рисках
□ Вести журналы (аварий, инструктажей, проверок)
□ Поддерживать порядок на рабочем месте
```

**Шаг 3. Завершение смены:**
```
□ Передать дежурство: текущие аварии, незакрытые заявки, важные сообщения
□ Проверить сохранность оборудования и документов
□ Зафиксировать окончание дежурства в журнале
□ При необходимости — сообщить руководителю о проблемах
```

#### **ЭТАП 3: Еженедельное/ежемесячное ТО и контроль**

**Шаг 4. Техническое обслуживание:**
```
□ Проверить работу системы записи разговоров, объем памяти
□ Протестировать ИБП: переключение, время автономной работы
□ Обновить ПО CRM/антивирусы при наличии обновлений
□ Проверить наличие расходников (бумага, картриджи)
□ Устранить мелкие неисправности или вызвать подрядчика
```

**Шаг 5. Контроль качества работы:**
```
□ Выборочно прослушать записи разговоров (10-20% от общего числа)
□ Оценить соблюдение скриптов, вежливость, полноту информации
□ Проверить своевременность регистрации и назначения заявок
□ Сформировать отчет по KPI за период
□ Провести разбор ошибок с диспетчерами
```

#### **ЭТАП 4: Квартальное/годовое обеспечение**

**Шаг 6. Работа с персоналом:**
```
□ Провести повторный инструктаж по охране труда и регламентам
□ Организовать обучение по новым нормативам/изменениям в CRM
□ Провести аттестацию диспетчеров на знание регламентов
□ Организовать медицинские осмотры (по графику)
□ Оценить KPI, применить меры мотивации/дисциплинарные
```

**Шаг 7. Актуализация документации:**
```
□ Проверить актуальность регламентов, инструкций, скриптов
□ Внести изменения при обновлении нормативной базы
□ Согласовать и утвердить новые редакции документов
□ Ознакомить персонал под подпись
□ Архивировать устаревшие версии
```

**Шаг 8. Отчетность и планирование:**
```
□ Сформировать квартальный/годовой отчет по работе АДС:
   - Количество принятых обращений по категориям
   - Соблюдение нормативов времени ответа/регистрации
   - Статистика аварий и сроков их устранения
   - Удовлетворенность заявителей
   - Выявленные проблемы и принятые меры
□ Представить отчет руководству УК
□ Скорректировать план работы АДС на следующий период
□ Запланировать закупку оборудования/обучение при необходимости
```

### 9. ДЕЙСТВИЯ ПРИ НАРУШЕНИИ ТРЕБОВАНИЙ К СОДЕРЖАНИЮ АДС

| Нарушение | Категория | Действия |
|-----------|-----------|----------|
| **Отсутствие дежурного диспетчера** | Критическое | Немедленное привлечение резерва, доклад руководству, служебное расследование |
| **Неисправность критического оборудования > 1 часа** | Критическое | Вызов подрядчика, переход на резервные каналы связи, информирование собственников |
| **Отсутствие записи разговоров** | Значительное | Технический аудит, устранение причины, дисциплинарные меры |
| **Непрохождение инструктажа персоналом** | Значительное | Отстранение от работы, проведение внепланового инструктажа |
| **Нарушение сроков регистрации заявок** | Значительное | Разбор с диспетчером, дополнительное обучение, корректировка нагрузки |
| **Неактуальные регламенты/скрипты** | Незначительное | Актуализация документов, ознакомление персонала |

**Последствия для УК при ненадлежащем содержании АДС:**
- Штрафы по договорам с собственниками/муниципалитетом
- Предписания ГЖИ, административная ответственность
- Жалобы собственников, снижение репутации
- Риски при авариях из-за несвоевременного реагирования

### 13. КРИТЕРИИ ПРИЁМКИ РАБОТ ПО СОДЕРЖАНИЮ АДС

| Критерий | Оценка "Выполнено" | Оценка "Не выполнено" |
|----------|-------------------|---------------------|
| **Доступность службы** | 24/7 без перерывов > 15 минут | Регулярные перерывы в работе |
| **Исправность оборудования** | ≥ 99% времени работы | Частые сбои, отсутствие резерва |
| **Укомплектованность штата** | 100% по графику смен | Дежурство с нарушениями графика |
| **Качество связи** | Разборчивость ≥ 90%, запись 100% звонков | Плохое качество, пропуски записи |
| **Документирование** | Полные журналы, актуальные регламенты | Отсутствие/неполнота документов |
| **Обучение персонала** | 100% прошли инструктажи и аттестацию | Наличие необученных сотрудников |
| **Отчетность** | Своевременные отчеты с анализом KPI | Отсутствие или формальная отчетность |
| **Взаимодействие с экстренными службами** | Отлаженные процедуры, совместные учения | Отсутствие согласованных алгоритмов |
